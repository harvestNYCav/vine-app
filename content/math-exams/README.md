# Released Math Exam Catalog

This directory contains the typed catalog for official NYSED released math questions. The active catalog is intentionally multiple-choice only and covers Grades 3–8 for 2013–2019 and 2021–2026; the archive has no 2020 release. Short-answer and constructed-response records are not imported or kept in a parallel hand-authored tree, so they cannot become unreachable duplicate content. Official question crops live in `public/nysed/math`.

`generated/catalog.json` and its matching WebPs are reproducible outputs of `scripts/import_nysed_math_mc.py`. Schema version 2 discovers source URLs from NYSED's English and Spanish indexes, validates item/key/crop parity, rejects answer-key leakage, and requires a substantive localized explanation for every question. Do not hand-edit generated records.

For the 228 multiple-choice questions from 2013–2014, the importer extracts the correct `Answer Choice` or `Answer Option` block from NYSED's own annotated extended rationale. The hash-pinned `official-rationale-overrides.json` ledger covers 97 audited records: 92 restore fractions, operators, exponents, equations, or page-layout artifacts lost by PDF text extraction while retaining `official-nysed` provenance, and 5 narrowly correct semantic mistakes with `official-nysed-corrected` provenance. The 92 extraction repairs comprise 21 Grade 3–4 records and 71 Grade 5–8 records. The importer refuses a ledger repair if the raw extracted rationale changes. The UI identifies the five semantic repairs as “Official NYSED rationale, corrected by Vine.” Those exams have no official Spanish edition, so the same English rationale is stored in both localized explanation fields; this is a data-shape requirement, not an unofficial translation.

For all 1,611 questions from 2015 onward, explanations are original Vine educational content with `vine-authored` provenance. They are checked in as per-exam sidecars under `explanations/`. Each explanation sidecar is pinned to the exact English and optional Spanish question-image bytes, the importer's source-extracted localized OCR alt, answer key, and standards. The importer validates that explanation input first. Only afterward does it replace the browser-facing OCR alt with the independently reviewed accessibility description, so explanation hashes do not depend on presentation wording. The importer fails closed if a sidecar is missing, incomplete, stale, non-canonical, or has the wrong provenance; regenerate and review sidecars whenever any pinned input changes.

All 1,839 Grade 3–8 questions have human-reviewed accessibility descriptions under `accessibility/`, covering 3,131 localized question crops. Grades 5–8 account for 1,277 questions and 2,174 of those localizations. Each record independently pins the exact English and optional Spanish crop bytes, question identity, language coverage, source-critical mathematical tokens, and accessibility policy. Strict loading requires exact exam coverage and rejects stale hashes, unreviewed sentinels, damaged extraction artifacts, bare or missing choices, admissions that content is cropped, cut off, garbled, or illegible, generic image fallbacks, and grading leaks. Source-PDF hashes and exact crop geometry also pin the narrow crop extensions, footer masks, and text overlays used to restore audited content that is clipped or malformed in an official release. These descriptions become the browser-facing alt text only after the source-OCR-pinned explanation has validated.

The Grade 6 Statistics lesson template is intentionally present even though no released Grade 6 multiple-choice item in the 13 imported exam years is coded to the Statistics and Probability domain. The source PDFs and item maps were audited for this absence; do not invent a question or reassign a question from another grade merely to populate that lesson.

Install the pinned Python packages from `scripts/requirements-nysed.txt` before rebuilding. Scanned editions also require Tesseract 5 on `PATH` (or an explicit `--tesseract` path). Run the importer with `--contact-sheets` and review those sheets before committing regenerated assets.

Keep answer keys server-only by passing browser-facing records through `toPublicMathExamQuestion`. `catalog-runtime.ts` is guarded by Next's `server-only` marker; `index.ts` exposes course metadata to Server Components and `lib/math-exams.ts` owns question lookup. `tests/math-exams.test.ts` verifies the complete inventory and every referenced asset.

Spanish source editions are available for 2017–2019 and 2021–2026. Releases from 2013–2016 intentionally remain English-only; do not create unofficial translations of copyrighted questions.

Treat an exam ID as an immutable content version. If question membership or point totals change, publish a new ID so historical best-score numerators and denominators remain comparable.
