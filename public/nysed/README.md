# NYSED Released Question Assets

These question-only WebP crops come from linked New York State Education Department released-item PDFs. They are included for Vine's noncommercial educational use and must be displayed with the in-app NYSED attribution and independent/non-endorsement disclaimer.

Do not add scoring-guide sample responses or rubric text here. Preserve legibility and record each image's exact dimensions in the generated catalog. Keep official Spanish crops only where NYSED publishes a direct Spanish edition; 2013–2016 remain English-only.

Math assets live under `math/{year}/grade-{grade}/{language}` and are rebuilt with `scripts/import_nysed_math_mc.py`.

ELA assets live under `ela/{year}/grade-{grade}/en` and contain only the released question and A–D choices. Full passages remain in the linked official PDF so the original third-party source and permission credits stay with them. ELA assets are rebuilt with `scripts/import_nysed_ela_mc.py`.
